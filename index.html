Chat Widget

Click on the chat icon to open the chat widget

Install Widget

To install techvill-widget, you can place this code before the <body/> tag on every page of your website.

<script
src="https://techvill-chatwidget.netlify.app/sdk.js"
data-iframe-src="https://techvill-chat.netlify.app/"
data-fab-icon="./assets/chat-outline.png"
data-fab-close-icon="./assets/close.png"
data-fab-bg-color="#2C2C2C"
data-iframe-height="632"
data-iframe-width="390"
>
</script>
