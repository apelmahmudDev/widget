Embed your website

Click on the FAB icon to open your website in the Widget iframe.

Install Widget

To install widget, you can place this code before the <body/> tag on every page of your website.

<script
src="https://techvill-chatwidget.netlify.app/chatbot-sdk.js"
data-iframe-src="https://chatbot-free.vercel.app/artifism/client-chatbot"
data-iframe-height="632"
data-iframe-width="390"
>
</script>
